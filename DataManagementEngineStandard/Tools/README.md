# Tools

Code generation and scaffolding utilities.

## Key Components
- `ClassCreator.*.cs`: core/advanced/database/web API/entity generation features.
- `WebApiGenerator.cs`: Web API scaffold generation.
- `Helpers/*`: focused generation helpers (POCO, DB classes, docs, UI components, validation/testing).

## How It Fits
Supports developer productivity by generating classes and scaffolding from schema/entity metadata.
