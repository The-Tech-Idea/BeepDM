# FileManager

File-based datasource and parsing utilities.

## Key Files
- `CSVDataSource.cs`: CSV datasource implementation.
- `CSVAnalyser.cs`: CSV schema/content analyzer.
- `CSVTypeMapper.cs`: CSV-to-type mapping helper.
- `ICSVDataReader.cs`: CSV reader abstractions.
- `TextFieldParser.cs`: text field parsing logic.

## How It Fits
Enables ingestion and querying of CSV/text data through Beep datasource abstractions.
