UnitOfWorkWrapper Class

? Constructor
public UnitOfWorkWrapper(object unitOfWork)

Initializes a new wrapper around a UnitOfWork instance.

? Dynamic Properties

The UnitOfWorkWrapper exposes all properties of the wrapped UnitOfWork instance dynamically. Key properties include:

  • Units: The entity collection
  • IsDirty: Change tracking status
  • PrimaryKey: Primary key field name
  • EntityName: Entity name
  • DataSource: Associated data source
? Dynamic Methods

The UnitOfWorkWrapper forwards all method calls to the wrapped UnitOfWork instance. All CRUD operations, navigation methods, and utility functions are available through the wrapper.
