## Objective

Ensure every `IProxyNode` in the cluster shares a single `ICircuitStateStore` and a
single `IProxyAuditSink`, so tripped circuits and audit records are visible
cluster-wide.  
Also implement `ApplyClusterPolicy(ProxyPolicy)` so one policy update propagates to
all current and future nodes atomically.

## 1. Shared-State Injection on `AddNode`

**File:** `ProxyCluster.NodeManagement.cs`

```csharp
public partial class ProxyCluster
{
    // ── Node management ───────────────────────────────────────────────────

    public void AddNode(IProxyNode node)
    {
        if (node is not ProxyNode concrete)
            throw new ArgumentException(
                $"Expected {nameof(ProxyNode)}; got {node?.GetType().Name}.");

        // Wire shared state to the underlying proxy so it uses the same
        // circuit-break store and audit sink as every other node in the cluster.
        WireSharedState(concrete);

        _nodes[concrete.NodeId] = concrete;

        // Rebuild deterministic router ring (no-op for non-consistent-hash)
        RebuildRouterIfNeeded();
    }

    public void RemoveNode(string nodeId)
    {
        if (_nodes.TryRemove(nodeId, out var node))
        {
            // No-op drain (caller should call DrainNode first for clean shutdown)
            node.IsDraining = true;
            node.Proxy?.Dispose();
            RebuildRouterIfNeeded();
        }
    }

    public IReadOnlyList<IProxyNode> GetNodes()
        => _nodes.Values.Cast<IProxyNode>().ToList().AsReadOnly();

    // ── Drain ─────────────────────────────────────────────────────────────

    public void DrainNode(string nodeId, int timeoutMs = 30_000)
    {
        if (!_nodes.TryGetValue(nodeId, out var node)) return;

        node.IsDraining = true;
        var deadline = DateTime.UtcNow.AddMilliseconds(timeoutMs);

        while (node.InFlightCount > 0 && DateTime.UtcNow < deadline)
            Thread.Sleep(50);
    }

    // ── Wiring helper ─────────────────────────────────────────────────────

    private void WireSharedState(ProxyNode node)
    {
        // Push the cluster-level shared state into the underlying IProxyDataSource.
        // This ensures:
        //   • Circuit breaks on node A are visible on node B.
        //   • All audit records land in the same sink.

        if (node.Proxy is ProxyDataSource pds)
        {
            // Replace the node's private state store with the cluster's shared one
            pds.CircuitStateStore = _circuitStateStore;
            // Replace the node's audit sink with the cluster-wide one
            pds.AuditSink = _auditSink;
        }
        // For non-ProxyDataSource implementations the shared state is already set
        // at construction; nothing more to do.
    }

    private void RebuildRouterIfNeeded()
    {
        if (_nodeRouter is ConsistentHashRouter chr)
            chr.Rebuild(_nodes.Values);
    }
}
```

## 2. `ApplyClusterPolicy`

**File:** `ProxyCluster.cs` (add to the main partial)

```csharp
public void ApplyClusterPolicy(ProxyPolicy policy)
{
    if (policy is null) throw new ArgumentNullException(nameof(policy));

    _clusterPolicy = policy;

    // Fan the same policy object out to every node's proxy.
    // The node-proxy performs its own internal reconfiguration (retries,
    // backpressure, watch-dog interval, etc.) via its ApplyPolicy() path.
    foreach (var n in _nodes.Values)
    {
        try   { n.Proxy.ApplyPolicy(policy); }
        catch (Exception ex)
        {
            // Policy fan-out should never crash the cluster — log and continue.
            _auditSink.Write(new ProxyAuditEntry
            {
                EntryKind   = AuditEntryKind.PolicyChange,
                NodeId      = n.NodeId,
                Success     = false,
                Message     = $"Policy fan-out failed: {ex.Message}",
                OccurredAtUtc = DateTime.UtcNow
            });
        }
    }

    // Re-apply the routing strategy in case NodeRoutingStrategy changed.
    _nodeRouter = BuildRouter(policy.NodeRoutingStrategy);

    // Rebuild consistent-hash ring with the new weights (if applicable).
    RebuildRouterIfNeeded();

    OnClusterPolicyChanged?.Invoke(this, new ClusterPolicyChangedEventArgs(policy));
}

// Must also apply shared state to nodes added AFTER a policy change.
// AddNode() always calls WireSharedState(), so newly-added nodes are always
// consistent without extra work here.
```

## 3. Router Factory

```csharp
private static INodeRouter BuildRouter(ProxyNodeRoutingStrategy strategy)
    => strategy switch
    {
        ProxyNodeRoutingStrategy.LeastConnections  => new LeastConnectionsRouter(),
        ProxyNodeRoutingStrategy.WeightedRoundRobin => new WeightedRoundRobinRouter(),
        ProxyNodeRoutingStrategy.LowestLatency      => new LowestLatencyRouter(),
        ProxyNodeRoutingStrategy.PrimaryWithStandby => new PrimaryWithStandbyRouter(),
        ProxyNodeRoutingStrategy.ConsistentHash     => new ConsistentHashRouter(),
        _                                           => new LeastConnectionsRouter()
    };
```

## 5. State Sharing Matrix

| Resource | Owned by | Shared via |
|----------|----------|-----------|
| `ICircuitStateStore` | `ProxyCluster` | Pushed to each node's `ProxyDataSource.CircuitStateStore` on `AddNode` |
| `IProxyAuditSink` | `ProxyCluster` | Pushed to each node's `ProxyDataSource.AuditSink` on `AddNode` |
| `ProxyPolicy` | `ProxyCluster` | Fan-out call to `node.Proxy.ApplyPolicy()` on every `ApplyClusterPolicy()` |
| `INodeRouter` | `ProxyCluster` | Single instance, accessed under no lock (router impls are thread-safe) |
| `ConnectionState` | Per-node `ProxyDataSource` | Read-only from cluster; cluster does not aggregate connection state |

## Acceptance Criteria

- [ ] `ProxyCluster.AddNode(node)` sets `node.Proxy.CircuitStateStore` to the cluster's shared store
- [ ] `ProxyCluster.AddNode(node)` sets `node.Proxy.AuditSink` to the cluster's shared sink
- [ ] `ProxyCluster.ApplyClusterPolicy(policy)` calls `node.Proxy.ApplyPolicy(policy)` for every registered node
- [ ] Fan-out failure in a single node logs an audit entry but does **not** abort the fan-out
- [ ] `DrainNode(id, timeout)` returns as soon as `InFlightCount == 0` or timeout elapses
- [ ] `RemoveNode(id)` marks the node as draining and calls `Dispose()` on its proxy
- [ ] Changing `NodeRoutingStrategy` in policy triggers a router rebuild
