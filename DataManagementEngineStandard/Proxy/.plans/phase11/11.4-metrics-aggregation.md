## 1. `GetClusterMetrics()`

Each `ProxyNode.Metrics` already calls `node.Proxy.GetMetrics()` under the hood
(as defined in Phase 11.1). The cluster-level method **merges** all nodes into a
single `IDictionary<string, DataSourceMetrics>` keyed by `DatasourceName`:

```csharp
public IDictionary<string, DataSourceMetrics> GetClusterMetrics()
{
    var result = new Dictionary<string, DataSourceMetrics>();

    foreach (var node in _nodes.Values)
    {
        var nodeMetrics = node.Proxy.GetMetrics();
        if (nodeMetrics is null) continue;

        foreach (var (dsName, metrics) in nodeMetrics)
        {
            if (!result.ContainsKey(dsName))
            {
                result[dsName] = metrics;
            }
            else
            {
                // Merge metrics: sum counters, weighted-average latency by TotalRequests
                result[dsName] = MergeMetrics(result[dsName], metrics);
            }
        }
    }

    // Also add per-node metrics under "node/<nodeId>" key
    foreach (var node in _nodes.Values)
    {
        var nm = node.Metrics;
        result[$"node/{node.NodeId}"] = nm;
    }

    return result;
}

private static DataSourceMetrics MergeMetrics(DataSourceMetrics a, DataSourceMetrics b)
{
    long totalA = a.TotalRequests;
    long totalB = b.TotalRequests;
    long total  = totalA + totalB;

    return new DataSourceMetrics
    {
        TotalRequests       = total,
        SuccessCount        = a.SuccessCount        + b.SuccessCount,
        ErrorCount          = a.ErrorCount          + b.ErrorCount,
        RetryCount          = a.RetryCount          + b.RetryCount,
        CircuitBreakerTrips = a.CircuitBreakerTrips + b.CircuitBreakerTrips,

        // Weighted-average latency
        AvgLatencyMs = total == 0
            ? 0
            : (a.AvgLatencyMs * totalA + b.AvgLatencyMs * totalB) / total,

        // P50 / P95 / P99 — take the max (worst case represents tail latency)
        P50LatencyMs = Math.Max(a.P50LatencyMs, b.P50LatencyMs),
        P95LatencyMs = Math.Max(a.P95LatencyMs, b.P95LatencyMs),
        P99LatencyMs = Math.Max(a.P99LatencyMs, b.P99LatencyMs)
    };
}
```

## 2. `GetClusterSloSnapshots()`

```csharp
public IReadOnlyList<ProxySloSnapshot> GetClusterSloSnapshots()
{
    var snapshots = new List<ProxySloSnapshot>();

    foreach (var node in _nodes.Values)
    {
        // Each ProxyDataSource exposes GetSloSnapshot(); call it directly.
        if (node.Proxy is ProxyDataSource pds)
        {
            var snap = pds.GetSloSnapshot();
            snapshots.Add(snap with { NodeId = node.NodeId });
        }
    }

    // Also add a cluster-aggregated row
    if (snapshots.Count > 0)
    {
        snapshots.Add(new ProxySloSnapshot
        {
            NodeId       = "cluster",
            WindowStart  = snapshots.Min(s => s.WindowStart),
            WindowEnd    = snapshots.Max(s => s.WindowEnd),
            TotalRequests = snapshots.Sum(s => s.TotalRequests),
            SuccessRate   = snapshots.Average(s => s.SuccessRate),
            AvgLatencyMs  = snapshots.Average(s => s.AvgLatencyMs),
            P95LatencyMs  = snapshots.Max(s => s.P95LatencyMs),
            P99LatencyMs  = snapshots.Max(s => s.P99LatencyMs),
            CircuitOpen   = snapshots.Any(s => s.CircuitOpen)
        });
    }

    return snapshots.AsReadOnly();
}
```

## 3. `ProxySloSnapshot` Record

If not already defined in `ProxyotherClasses.cs`, add:

```csharp
/// <summary>
/// Rolling-window SLO snapshot for one node or the entire cluster.
/// </summary>
public record ProxySloSnapshot
{
    /// <summary>"cluster" or a node ID.</summary>
    public string   NodeId         { get; init; }
    public DateTime WindowStart    { get; init; }
    public DateTime WindowEnd      { get; init; }
    public long     TotalRequests  { get; init; }
    public double   SuccessRate    { get; init; }   // 0.0–1.0
    public double   AvgLatencyMs   { get; init; }
    public double   P95LatencyMs   { get; init; }
    public double   P99LatencyMs   { get; init; }
    public bool     CircuitOpen    { get; init; }
}
```

## 4. Metrics Reporting Dashboard Pattern

Consumers can use these APIs to drive a simple dashboard without needing to know
cluster internals:

```csharp
var cluster = (IProxyCluster)dme.GetDataSource("orders-cluster");

// Overall cluster health
var metrics  = cluster.GetClusterMetrics();
var clusterM = metrics["cluster"];   // aggregated
Console.WriteLine($"Total RPS: {clusterM.TotalRequests / windowSec:N0}");
Console.WriteLine($"Avg latency: {clusterM.AvgLatencyMs:N1}ms");

// Per-node health
foreach (var node in cluster.GetNodes())
{
    var nm = metrics[$"node/{node.NodeId}"];
    Console.WriteLine(
        $"{node.NodeId} [{(node.IsAlive ? "UP" : "DOWN")}] " +
        $"in-flight: {node.InFlightCount}  " +
        $"p99: {nm.P99LatencyMs:N1}ms  " +
        $"errors: {nm.ErrorCount}");
}

// SLO window
foreach (var snap in cluster.GetClusterSloSnapshots())
{
    Console.WriteLine(
        $"{snap.NodeId,-15} success={snap.SuccessRate:P1}  " +
        $"p95={snap.P95LatencyMs:N1}ms  circuit={(snap.CircuitOpen ? "OPEN" : "closed")}");
}
```

## 5. Data Flow Diagram

```
cluster.GetClusterMetrics()
  │
  ├── node-1.Proxy.GetMetrics() → { "ordersDB": {Req:500, Err:2, AvgLat:12ms} }
  ├── node-2.Proxy.GetMetrics() → { "ordersDB": {Req:480, Err:1, AvgLat:15ms} }
  ├── node-3.Proxy.GetMetrics() → { "ordersDB": {Req:520, Err:0, AvgLat:10ms} }
  │
  └── MergeMetrics(...)
        ├── "ordersDB" → {Req:1500, Err:3, AvgLat:12.3ms (weighted), P99:max(...)}
        ├── "node/node-1" → per-node copy
        ├── "node/node-2" → per-node copy
        └── "node/node-3" → per-node copy
```

## Acceptance Criteria

- [ ] `GetClusterMetrics()` returns an entry for every node (`node/<nodeId>`)
- [ ] Aggregated latency is the weighted average by `TotalRequests`, not a simple mean
- [ ] P95/P99 latency takes the **max** across nodes (conservative / correct)
- [ ] `GetClusterSloSnapshots()` includes a `"cluster"` row
- [ ] `"cluster"` row `CircuitOpen` is `true` if **any** node is open
- [ ] No two calls mutate shared state (methods are read-only)
