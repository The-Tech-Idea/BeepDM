## Objective

1. **Consistent hashing** — complete the `ConsistentHashRouter` introduced in Phase 11.1
   with a full MurmurHash3 implementation, virtual-slot ring, and minimal remap
   behaviour on topology changes.

2. **Connection multiplexing** — cap the total number of backend connections the cluster
   opens across all nodes, queue excess requests, and timeout the queue.

## 2. Complete `ConsistentHashRouter`

**File:** `ProxyCluster.NodeRouting.cs`

```csharp
internal sealed class ConsistentHashRouter : INodeRouter
{
    // ── Ring state ────────────────────────────────────────────────────
    private volatile SortedDictionary<uint, ProxyNode> _ring
        = new SortedDictionary<uint, ProxyNode>();
    private const int VirtualSlots = 150;   // matches Redis Cluster convention

    // ── Public rebuild (called by AddNode / RemoveNode / ApplyClusterPolicy) ──
    internal void Rebuild(IEnumerable<ProxyNode> nodes)
    {
        var newRing = new SortedDictionary<uint, ProxyNode>();
        foreach (var node in nodes)
        {
            for (int v = 0; v < VirtualSlots; v++)
            {
                // Use "nodeId:v" as the hash input for each virtual slot
                uint slot = MurmurHash3(Encoding.UTF8.GetBytes($"{node.NodeId}:{v}"));
                // Collision: last writer wins (acceptable — extremely rare with 32-bit hash)
                newRing[slot] = node;
            }
        }
        _ring = newRing;   // atomic reference swap (volatile write)
    }

    // ── Routing ───────────────────────────────────────────────────────
    public ProxyNode SelectBest(IReadOnlyList<ProxyNode> candidates, ProxyExecutionContext? ctx)
    {
        var ring = _ring;
        if (ring.Count == 0)
            return candidates[0];

        // Hash key: prefer SessionKey > CorrelationId > random
        string hashKey = ctx?.SessionKey
            ?? ctx?.CorrelationId
            ?? Guid.NewGuid().ToString();

        uint targetSlot = MurmurHash3(Encoding.UTF8.GetBytes(hashKey));

        // Clockwise walk: first ring entry with key >= targetSlot
        ProxyNode? selected = null;
        foreach (var kv in ring)
        {
            if (kv.Key >= targetSlot && candidates.Contains(kv.Value))
            {
                selected = kv.Value;
                break;
            }
        }

        // Wrap around (target slot > all ring slots)
        if (selected is null)
        {
            foreach (var kv in ring)
            {
                if (candidates.Contains(kv.Value))
                {
                    selected = kv.Value;
                    break;
                }
            }
        }

        return selected ?? candidates[0];
    }

    // ── MurmurHash3 (32-bit, x86) ─────────────────────────────────────
    // Reference: Austin Appleby's public-domain MurmurHash3
    internal static uint MurmurHash3(ReadOnlySpan<byte> data, uint seed = 0)
    {
        uint h1  = seed;
        uint c1  = 0xcc9e2d51;
        uint c2  = 0x1b873593;
        int  len = data.Length;
        int  pos = 0;

        // Body — process in 4-byte blocks
        int blocks = len / 4;
        for (int i = 0; i < blocks; i++, pos += 4)
        {
            uint k1 = BinaryPrimitives.ReadUInt32LittleEndian(data.Slice(pos, 4));
            k1 *= c1;
            k1  = RotateLeft(k1, 15);
            k1 *= c2;
            h1 ^= k1;
            h1  = RotateLeft(h1, 13);
            h1  = h1 * 5 + 0xe6546b64;
        }

        // Tail — remaining bytes
        uint tail = 0;
        int remaining = len & 3;
        if (remaining >= 3) tail ^= (uint)data[pos + 2] << 16;
        if (remaining >= 2) tail ^= (uint)data[pos + 1] << 8;
        if (remaining >= 1)
        {
            tail ^= data[pos];
            tail *= c1;
            tail  = RotateLeft(tail, 15);
            tail *= c2;
            h1   ^= tail;
        }

        // Finalization (fmix32)
        h1 ^= (uint)len;
        h1 ^= h1 >> 16;
        h1 *= 0x85ebca6b;
        h1 ^= h1 >> 13;
        h1 *= 0xc2b2ae35;
        h1 ^= h1 >> 16;

        return h1;
    }

    private static uint RotateLeft(uint value, int bits)
        => (value << bits) | (value >> (32 - bits));
}
```

## 3. Ring Rebalance on Topology Change

**Only 1/N of keys remap** when one node is added or removed — the main advantage
over consistent hashing.

```csharp
// In ProxyCluster.NodeManagement.cs — already called by AddNode / RemoveNode:
private void RebuildRouterIfNeeded()
{
    if (_nodeRouter is ConsistentHashRouter chr)
        chr.Rebuild(_nodes.Values.Where(n => n.IsAlive));
}
```

When a node goes down, `MarkNodeDead` calls `RebuildRouterIfNeeded()` so the ring
no longer includes dead virtual slots, preventing continued routing to a dead node.

## 4. Connection Multiplexing (Backend Connection Quota)

**File:** `ProxyCluster.cs`

```csharp
// ── Backend connection semaphore ──────────────────────────────────────────
private SemaphoreSlim? _backendConnectionSemaphore;
private int            _queueDepth;   // current queue depth counter

private void InitConnectionMultiplexing(ProxyPolicy policy)
{
    _backendConnectionSemaphore?.Dispose();
    _backendConnectionSemaphore = null;

    if (policy.ClusterMaxBackendConnections > 0)
        _backendConnectionSemaphore = new SemaphoreSlim(
            initialCount: policy.ClusterMaxBackendConnections,
            maxCount:     policy.ClusterMaxBackendConnections);
}
```

Per-operation connection gate (wraps every delegated operation):

```csharp
/// <summary>
/// Acquires a backend connection slot.  Returns an IDisposable that releases
/// the slot on Dispose, suitable for use in a using statement.
/// </summary>
private async ValueTask<IDisposable> AcquireBackendSlotAsync(CancellationToken ct)
{
    var sem = _backendConnectionSemaphore;
    if (sem is null)
        return NullDisposable.Instance;   // no quota — pass through immediately

    // Check queue depth guard
    int current = Interlocked.Increment(ref _queueDepth);
    if (current > _clusterPolicy.ClusterMaxQueueDepth)
    {
        Interlocked.Decrement(ref _queueDepth);
        throw new ProxyQueueDepthExceededException(
            $"Cluster queue depth exceeded " +
            $"(max {_clusterPolicy.ClusterMaxQueueDepth}).");
    }

    bool acquired = await sem
        .WaitAsync(_clusterPolicy.ClusterQueueTimeoutMs, ct)
        .ConfigureAwait(false);

    Interlocked.Decrement(ref _queueDepth);

    if (!acquired)
        throw new ProxyQueueTimeoutException(
            $"Timed out waiting for a backend connection slot " +
            $"after {_clusterPolicy.ClusterQueueTimeoutMs}ms.");

    return new SemaphoreReleaser(sem);
}

private sealed class SemaphoreReleaser : IDisposable
{
    private readonly SemaphoreSlim _sem;
    public SemaphoreReleaser(SemaphoreSlim sem) { _sem = sem; }
    public void Dispose() => _sem.Release();
}

private sealed class NullDisposable : IDisposable
{
    public static readonly NullDisposable Instance = new();
    public void Dispose() { }
}
```

Exception types:

Usage in a delegated async operation:

```csharp
public async Task<IEnumerable<object>> GetEntityAsync(
    string entityName, List<AppFilter> filter, CancellationToken ct = default)
{
    var node = SelectNode("GetEntity", entityName, null);
    await ApplyFaultInjectionAsync("GetEntity", entityName, node, ct);

    await using var slot = await AcquireBackendSlotAsync(ct);

    node.IncrementInFlight();
    try
    {
        var result = await node.Proxy.GetEntityAsync(entityName, filter).ConfigureAwait(false);
        RecordNodeOutcome(node, true);
        return result;
    }
    catch (Exception ex) when (ex is not OperationCanceledException)
    {
        RecordNodeOutcome(node, false);
        throw;
    }
    finally
    {
        node.DecrementInFlight();
    }
}
```

## 6. Sequence Diagram — Consistent Hash Ring

```
AddNode("node-1"), AddNode("node-2"), AddNode("node-3")
  │
  ▼
ConsistentHashRouter.Rebuild([node-1, node-2, node-3])
  ├─ 150 virtual slots for node-1 placed at hash("node-1:0") .. hash("node-1:149")
  ├─ 150 virtual slots for node-2
  └─ 150 virtual slots for node-3
     → ring has 450 entries

GetEntity(ctx={CorrelationId="req-abc"})
  │
  ├─ MurmurHash3("req-abc") → 0x7A3F1C22
  ├─ Walk ring clockwise from 0x7A3F1C22
  └─ First alive entry → node-2 slot at 0x7A401000 → route to node-2

RemoveNode("node-2")
  │
  ├─ ConsistentHashRouter.Rebuild([node-1, node-3])
  │     → ring now has 300 entries  (node-2's 150 slots removed)
  │
GetEntity(ctx={CorrelationId="req-abc"})   (same request again)
  │
  ├─ MurmurHash3("req-abc") → 0x7A3F1C22  (same hash)
  ├─ Walk ring → next alive slot after 0x7A3F1C22
  └─ node-1 slot at 0x7A501000 → route to node-1

Only requests whose hash key landed between 0x7A401000 and 0x7A501000 were remapped.
All other requests still route to their original node.
```

## 7. Sequence Diagram — Connection Quota

```
Cluster: ClusterMaxBackendConnections=3

At steady state: 3 concurrent in-flight requests consuming all 3 slots.

4th request arrives:
  │
  ├─ AcquireBackendSlotAsync()
  │     ├─ _queueDepth = 1  (within ClusterMaxQueueDepth)
  │     ├─ SemaphoreSlim.WaitAsync(3000ms)
  │     │
  │     │  ... 800ms later: one of the 3 in-flight requests completes ...
  │     │      slot released via SemaphoreReleaser.Dispose()
  │     │
  │     └─ 4th request acquires slot → proceeds

5th request, all 3 slots busy for > 3000ms:
  │
  ├─ AcquireBackendSlotAsync()
  │     └─ SemaphoreSlim.WaitAsync(3000ms) → false
  └─ ProxyQueueTimeoutException thrown
```

## Acceptance Criteria

- [ ] `ConsistentHashRouter` with 3 nodes creates exactly 450 ring entries
- [ ] Same `CorrelationId` routes to the same node repeatedly (deterministic)
- [ ] After removing node-2, routing to "surviving" hashes is unchanged (minimal remap)
- [ ] Ring rebuild is thread-safe (volatile reference swap)
- [ ] `ClusterMaxBackendConnections = 0` disables quota (no semaphore, no overhead)
- [ ] Requests queue when all slots are busy (up to `ClusterMaxQueueDepth`)
- [ ] Queue depth exceeded → `ProxyQueueDepthExceededException` immediately
- [ ] Queue wait exceeds `ClusterQueueTimeoutMs` → `ProxyQueueTimeoutException`
- [ ] `SemaphoreReleaser.Dispose()` always called in finally (slot never leaked)
- [ ] `ApplyClusterPolicy` rebuilds semaphore with new `ClusterMaxBackendConnections`
