# Phase 11.6 — Entity-Affinity Routing

**Priority:** P2  
**Depends on:** Phase 11.1, Phase 11.3  
**Goals satisfied:** G8 (entity-affinity / partition leadership)  
**Inspiration:** Kafka consumer-group leader assignment, Vitess VSchema shard-to-tablet mapping  
**Estimated effort:** ~0.5 day

---

## Objective

Allow the operator to declare that all reads/writes for a given entity (table name)
or entity-name prefix should always route to a specific node.  
This is the database equivalent of Kafka partition leadership — one primary "owns" a
logical shard, so hot-path reads avoid cross-partition scatter.

/// What the cluster does when the affinity-pinned node is unavailable.
/// </summary>
public enum EntityAffinityFallback
{
    /// <summary>Use the normal router strategy to pick any live node.</summary>
    RouteToAny,
    /// <summary>Throw an exception — the entity cannot be served without its owner.</summary>

}

## 3. `EntityAffinityMap`

**File:** `EntityAffinityMap.cs`

```csharp
namespace TheTechIdea.Beep.Proxy
{
    /// <summary>
    /// Thread-safe map from entity (table) name → owning node ID.
    /// Supports exact-match and prefix-match lookups.
    /// Inspired by Vitess VSchema shard/tablet routing and Kafka partition leader election.
    /// </summary>
    public sealed class EntityAffinityMap
    {
        // Exact match: "Orders" → "node-1"
        private readonly ConcurrentDictionary<string, string> _exact
            = new(StringComparer.OrdinalIgnoreCase);

        // Prefix match (sorted for longest-prefix wins)
        // "Orders_" → "node-2"
        private readonly ConcurrentDictionary<string, string> _prefix
            = new(StringComparer.OrdinalIgnoreCase);

        /// <summary>
        /// When no exact/prefix match exists, fall back to this node.
        /// Null means "no static fallback — use the cluster router".
        /// </summary>
        public string? FallbackNodeId { get; set; }

        // ── Mutation ──────────────────────────────────────────────────

        /// <summary>Map an exact entity name to a node.</summary>
        public void MapEntity(string entityName, string nodeId)
        {
            ArgumentNullException.ThrowIfNullOrEmpty(entityName);
            ArgumentNullException.ThrowIfNullOrEmpty(nodeId);
            _exact[entityName] = nodeId;
        }

        /// <summary>Map all entity names starting with <paramref name="prefix"/> to a node.</summary>
        public void MapPrefix(string prefix, string nodeId)
        {
            ArgumentNullException.ThrowIfNullOrEmpty(prefix);
            ArgumentNullException.ThrowIfNullOrEmpty(nodeId);
            _prefix[prefix] = nodeId;
        }

        /// <summary>Remove exact mapping.</summary>
        public void UnmapEntity(string entityName) => _exact.TryRemove(entityName, out _);

        /// <summary>Remove prefix mapping.</summary>
        public void UnmapPrefix(string prefix) => _prefix.TryRemove(prefix, out _);

        // ── Lookup ────────────────────────────────────────────────────

        /// <summary>
        /// Resolve the preferred nodeId for <paramref name="entityName"/>.
        /// Returns null when no mapping is found and FallbackNodeId is null.
        /// </summary>
        public string? Resolve(string entityName)
        {
            if (string.IsNullOrEmpty(entityName)) return FallbackNodeId;

            // 1. Exact match
            if (_exact.TryGetValue(entityName, out var nodeId))
                return nodeId;

            // 2. Longest matching prefix wins
            string? bestPrefix = null;
            string? bestNodeId = null;
            foreach (var kv in _prefix)
            {
                if (entityName.StartsWith(kv.Key, StringComparison.OrdinalIgnoreCase))
                {
                    if (bestPrefix is null || kv.Key.Length > bestPrefix.Length)
                    {
                        bestPrefix = kv.Key;
                        bestNodeId = kv.Value;
                    }
                }
            }
            if (bestNodeId is not null) return bestNodeId;

            // 3. FallbackNodeId (may be null)
            return FallbackNodeId;
        }

        // ── Rebalancing ───────────────────────────────────────────────

        /// <summary>
        /// Reassign all entity/prefix mappings that point to <paramref name="failedNodeId"/>
        /// to a new node chosen from <paramref name="liveNodeIds"/>.
        /// Returns the reassignment map for event emission.
        /// </summary>
        public IReadOnlyDictionary<string, string> ReassignAfterFailure(
            string failedNodeId,
            IReadOnlyList<string> liveNodeIds)
        {
            if (liveNodeIds.Count == 0)
                return new Dictionary<string, string>();

            var reassignments = new Dictionary<string, string>(StringComparer.OrdinalIgnoreCase);

            // Round-robin across live nodes for spread
            int idx = 0;
            string NextLive() => liveNodeIds[idx++ % liveNodeIds.Count];

            foreach (var key in _exact.Keys.ToList())
                if (_exact.TryGetValue(key, out var nid) && nid == failedNodeId)
                {
                    var newNode = NextLive();
                    _exact[key] = newNode;
                    reassignments[$"entity:{key}"] = newNode;
                }

            foreach (var key in _prefix.Keys.ToList())
                if (_prefix.TryGetValue(key, out var nid) && nid == failedNodeId)
                {
                    var newNode = NextLive();
                    _prefix[key] = newNode;
                    reassignments[$"prefix:{key}"] = newNode;
                }

            if (FallbackNodeId == failedNodeId)
            {
                FallbackNodeId = liveNodeIds[0];
                reassignments["fallback"] = FallbackNodeId;
            }

            return reassignments;
        }

        // ── Snapshot ─────────────────────────────────────────────────

        public IReadOnlyDictionary<string, string> Snapshot()
        {
            var snap = new Dictionary<string, string>(StringComparer.OrdinalIgnoreCase);
            foreach (var kv in _exact)   snap[$"entity:{kv.Key}"]  = kv.Value;
            foreach (var kv in _prefix)  snap[$"prefix:{kv.Key}"]  = kv.Value;
            if (FallbackNodeId is not null) snap["fallback"]        = FallbackNodeId;
            return snap;
        }
    }
}
```

## 4. `SelectNode` — Entity-Affinity Branch

Add to `ProxyCluster.SelectNode()` before the normal routing path:

```csharp
private ProxyNode SelectNode(string operationName, string entityHint, ProxyExecutionContext ctx)
{
    var live = _nodes.Values
        .Where(n => n.IsAlive && !n.IsDraining)
        .ToList();

    if (live.Count == 0)
        throw new InvalidOperationException("ProxyCluster: no live nodes available.");

    // ── Entity-affinity lookup ─────────────────────────────────────────
    var affinityMap = _clusterPolicy.EntityAffinity;
    if (affinityMap is not null && !string.IsNullOrEmpty(entityHint))
    {
        string? preferredNodeId = affinityMap.Resolve(entityHint);
        if (preferredNodeId is not null)
        {
            var preferredNode = live.FirstOrDefault(n => n.NodeId == preferredNodeId);
            if (preferredNode is not null)
            {
                ApplySlowStartWeights(live);
                return preferredNode;
            }

            // Preferred node is dead / draining
            switch (_clusterPolicy.AffinityFallback)
            {
                case EntityAffinityFallback.ThrowException:
                    throw new InvalidOperationException(
                        $"Entity '{entityHint}' owner node '{preferredNodeId}' is unavailable " +
                        $"and AffinityFallback = ThrowException.");

                case EntityAffinityFallback.WaitForOwner:
                    // Brief wait for the node to recover — not more than one probe interval
                    Thread.Sleep(_clusterPolicy.NodeProbeIntervalMs);
                    preferredNode = live.FirstOrDefault(n => n.NodeId == preferredNodeId);
                    if (preferredNode is not null) return preferredNode;
                    // Still dead → fall through to RouteToAny
                    goto case EntityAffinityFallback.RouteToAny;

                case EntityAffinityFallback.RouteToAny:
                default:
                    break; // fall through to normal routing
            }
        }
    }

    // ── Session affinity (Phase 11.3) ──────────────────────────────────
    // ... existing session affinity code ...

    // ── Normal routing ──────────────────────────────────────────────────
    ApplySlowStartWeights(live);
    return _nodeRouter.SelectBest(live, ctx);
}
```

## 6. Configuration Example

```csharp
var affinityMap = new EntityAffinityMap();
affinityMap.MapEntity("Orders",    "node-1");   // all "Orders" ops → node-1
affinityMap.MapEntity("Inventory", "node-1");   // "Inventory"     → node-1
affinityMap.MapPrefix("Audit_",    "node-2");   // Audit_* tables  → node-2
affinityMap.FallbackNodeId = "node-1";          // everything else → node-1

var clusterPolicy = new ProxyPolicy
{
    NodeRoutingStrategy = ProxyNodeRoutingStrategy.LeastConnections,
    EntityAffinity      = affinityMap,
    AffinityFallback    = EntityAffinityFallback.RouteToAny
};

var cluster = new ProxyCluster(clusterPolicy);
cluster.AddNode(new ProxyNode("node-1", proxy1, weight: 2, role: ProxyDataSourceRole.Primary));
cluster.AddNode(new ProxyNode("node-2", proxy2, weight: 1, role: ProxyDataSourceRole.Replica));
```

## 7. Data Flow / Sequence Diagram

```
GetEntity("Orders", ...)
  │
  ▼
ProxyCluster.SelectNode(operationName: "GetEntity", entityHint: "Orders", ctx)
  │
  ├─ affinityMap.Resolve("Orders") → "node-1"
  │
  ├─ live nodes include "node-1"? ─YES─▶ return node-1
  │
  └─ NO (node-1 draining or dead)
        │
        AffinityFallback = RouteToAny ─▶ _nodeRouter.SelectBest(liveNodes) → node-2
        AffinityFallback = ThrowException ─▶ InvalidOperationException
        AffinityFallback = WaitForOwner ─▶ Thread.Sleep(probe interval) → retry → node-2

------
node-1 dies
  │
  ▼
MarkNodeDead("node-1")
  │── evict session affinity keys
  │── affinityMap.ReassignAfterFailure("node-1", ["node-2"])
  │     ├── _exact["Orders"]    → "node-2"
  │     └── _exact["Inventory"] → "node-2"
  │── OnAffinityRebalanced.Invoke( { "entity:Orders": "node-2", "entity:Inventory": "node-2" } )
```

## Acceptance Criteria

- [ ] `EntityAffinityMap.Resolve("Orders")` returns the mapped node ID
- [ ] Longest prefix wins when both a short and a long prefix match
- [ ] Case-insensitive lookup (`"orders"` == `"Orders"`)
- [ ] `SelectNode` routes to the affinity-mapped node when it is alive
- [ ] `AffinityFallback.RouteToAny` falls through to normal router when pinned node is dead
- [ ] `AffinityFallback.ThrowException` throws with a descriptive message
- [ ] `ReassignAfterFailure` reassigns all exact and prefix entries pointing at the failed node
- [ ] `OnAffinityRebalanced` fires with the correct reassignment dictionary
- [ ] Rebalancing is round-robin across live nodes (no single-node saturation)
