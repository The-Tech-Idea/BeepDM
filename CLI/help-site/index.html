üìñ Overview

BeepDM CLI is a powerful command-line interface for the BeepDM data management platform. It provides comprehensive tools for:

  • ‚úÖ Database and data source management
  • ‚úÖ Code generation (POCOs, Web APIs, Repositories, etc.)
  • ‚úÖ ETL operations and data synchronization
  • ‚úÖ Field mapping and schema analysis
  • ‚úÖ Multi-environment configuration management
  • ‚úÖ Driver and connection management
üí° New in this version: Three new batch code generation commands for generating multiple classes at once!

‚öôÔ∏è Installation

Install as .NET Tool

# Install from source
dotnet pack
dotnet tool install --global --add-source ./nupkg TheTechIdea.Beep.CLI

# Or install from NuGet (once published)
dotnet tool install --global TheTechIdea.Beep.CLI

Build from Source

cd CLI
dotnet build
dotnet run -- --help

Verify Installation

beep --version

üìã Workflows

Workflow 1: Schema Migration

  1. Compare schemas: beep dm compare-schemas DevDB ProdDB
  2. Export schema: beep dm export-schema ProdDB prod-schema.json
  3. Copy structure: beep etl copy-structure DevDB ProdDB Users
  4. Validate: beep etl validate DevDB ProdDB Users
  5. Copy data: beep etl copy-data DevDB ProdDB Users

Workflow 2: Multi-Environment Setup

  1. Create profiles: beep profile create dev, beep profile create prod
  2. Configure connections: beep config connection add
  3. Install drivers: beep driver scan ./ConnectionDrivers
  4. Test connections: beep ds test MyDB
  5. Generate code: beep class generate-poco-batch MyDB --output ./Models

ü§ù Contributing

Contributions are welcome! The CLI is designed to be extensible.

To add a new command:

  1. Create a new command class in Commands/
  2. Implement the Build() method returning a Command
  3. Use CliHelper for consistent messaging
  4. Register it in Program.cs
  5. Update README and this documentation
