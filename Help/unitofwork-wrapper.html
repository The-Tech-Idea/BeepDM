Overview

UnitOfWorkWrapper provides a strongly-typed interface around dynamically-created UnitOfWork instances. It implements both IUnitOfWorkWrapper and IUnitofWork interfaces with:

  • Safe property access - All property get/set operations are wrapped in try-catch
  • Event forwarding - Forwards events from underlying UnitOfWork via reflection
  • Oracle Forms compatibility - Navigation methods (MoveFirst, MoveNext, etc.)
  • Validation support - Integration with validation framework
  • Undo/Redo - Full undo/redo capability
  • Virtual loading - Support for large datasets with paging

Error Handling

All operations are wrapped in safe execution helpers:

  • GetPropertySafely - Returns default value on error
  • SetPropertySafely - Throws on error (setter failures are critical)
  • ExecuteSafely - Returns default value on error
  • ExecuteSafelyAsync - Returns default value on async error

ObjectDisposedException is thrown if wrapper is used after Dispose().

Usage Examples

Basic CRUD

var wrapper = UnitOfWorkFactory.CreateUnitOfWork(
    typeof(Customer), editor, "MyDb", "Customers", "Id");

// Add
wrapper.New();
wrapper.CurrentItem.Name = "John";
await wrapper.InsertAsync(wrapper.CurrentItem);

// Update
wrapper.MoveFirst();
wrapper.CurrentItem.Name = "Jane";
await wrapper.UpdateAsync(wrapper.CurrentItem);

// Commit
var result = await wrapper.Commit();
if (result.Flag == Errors.Ok)
    Console.WriteLine("Success");

Navigation with Events

wrapper.CurrentChanged += (s, e) => 
    Console.WriteLine($"Current: {wrapper.CurrentIndex}");

wrapper.PreCommit += (s, e) =>
{
    if (wrapper.GetInvalidItems().Any())
        e.Cancel = true;
};

wrapper.MoveFirst();
while (!wrapper.IsAtLast)
{
    Console.WriteLine(wrapper.CurrentItem.Name);
    wrapper.MoveNext();
}

Validation

wrapper.IsAutoValidateEnabled = true;
wrapper.BlockCommitOnValidationError = true;

var validation = wrapper.ValidateAll();
if (!validation.IsValid)
{
    foreach (var error in validation.Errors)
        Console.WriteLine(error.Message);
}
