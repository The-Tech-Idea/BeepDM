FilterHelper

Internal static class. Converts AppFilter collections to DataView.RowFilter expressions or evaluates them as LINQ predicates.

internal static class FilterHelper
{
    // Apply to DataView (string-based RowFilter)
    internal static DataView ApplyFilters(DataTable records, List<AppFilter> filters);

    // Apply as LINQ enumerable filter
    internal static IEnumerable<DataRow> ApplyFiltersEnumerable(DataTable records, List<AppFilter> filters);
}

Supported operators (both paths): =, contains, startswith, endswith, in, not in, >, <, >=, <=, <>, between, not between, is null, is not null.

The in/not in operators split collection values on ,;| and handle typed comparison (DateTime, bool, numeric, string).

FileDataSourceHelper

Internal static class. Creates ConnectionProperties for file-based data sources by matching file extensions to registered drivers.

internal static class FileDataSourceHelper
{
    // Bulk: create connections for a batch of files
    internal static List<ConnectionProperties> CreateFileConnections(IDMEEditor dme, string[] filenames);

    // Single: create a connection for one file
    // Matches by extension → driver config → DataSourceType → ConnectionString
    internal static ConnectionProperties CreateFileDataConnection(IDMEEditor dme, string file);

    // Driver matching logic:
    // 1. File extension → ConnectionDriversConfig.extensionstoHandle
    // 2. Driver config → DataSourceType + driver-specific connection string
    // 3. Sets DatabaseType, ConnectionName (from filename), Category = FILE
    // 4. Registers the connection in ConfigEditor if not already present
}

ReflectionHelper

Internal static class. Performance-oriented reflection utilities with Expression-compiled activator delegates.

internal static class ReflectionHelper
{
    // Expression-compiled constructor activator (avoids Activator.CreateInstance overhead)
    internal static Util.ObjectActivator<T> GetActivator<T>(ConstructorInfo ctor);

    // Generic instance creation
    internal static T CreateInstance<T>(params object[] args);

    // Type introspection
    internal static bool IsObjectNumeric(object o);
    internal static TypeCode GetTypeCode(Type type);
    internal static Type GetTypeFromStringValue(string str);
    internal static Type GetListType(object someList);

    // Relationship detection
    internal static bool HasNavigationProperties(Type type);
    internal static List<Type> GetNavigationPropertyTypes(Type type);
    internal static bool IsCollectionType(PropertyInfo prop);
}

ObjectActivator<T> delegate: Expression tree compiles a Func<object[], T> that directly invokes the constructor with typed parameter casting — significantly faster than Activator.CreateInstance for hot-path scenarios.
