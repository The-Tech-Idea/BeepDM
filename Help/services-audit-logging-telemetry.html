Architecture

TelemetryPipeline (shared orchestrator)
└── BoundedChannelQueue (backpressure-aware)
    └── BatchWriter (fan-out to sinks)

BeepLog (IBeepLog) ───────────────┐
BeepAudit (IBeepAudit) ────────────┤──→ TelemetryPipeline ──→ Sinks
Extensions (Audit/Log/Diagnostics) ┘

Pipeline stages (per envelope):
1. Enricher     → add correlation/context
2. Deduper      → suppress duplicates
3. Sampler      → probabilistic/rate-based sampling
4. Redactor     → PII/secret scrubbing
5. RateLimiter  → token bucket throttling
6. BatchWriter  → fan-out to sinks

Usage Examples

Setup Audit and Logging

var builder = Host.CreateApplicationBuilder();

builder.Services.AddBeepForDesktop(opts =>
{
    opts.AppRepoName = "MyApp";
    opts.DirectoryPath = AppContext.BaseDirectory;
});

// Add audit with tamper-evidence
builder.Services.AddBeepAudit(auditOpts =>
{
    auditOpts.Enabled = true;
    auditOpts.HashChain = true;
    auditOpts.Retention = new RetentionOptions { Days = 365 };
});

// Add structured logging
builder.Services.AddBeepLogging(logOpts =>
{
    logOpts.Enabled = true;
    logOpts.MinLevel = BeepLogLevel.Information;
});

var host = builder.Build();

Recording Audit Events

var audit = serviceProvider.GetRequiredService<IBeepAudit>();

await audit.RecordAsync("PipelineModified", "admin",
    "customer-sync",
    new Dictionary<string, object>
    {
        ["OldBatchSize"] = 100,
        ["NewBatchSize"] = 500,
        ["Timestamp"] = DateTime.UtcNow
    });

// Verify integrity
bool valid = audit.VerifyIntegrity("entry-id-123");

Structured Logging

var log = serviceProvider.GetRequiredService<IBeepLog>();

log.Info("Pipeline started");
log.Debug($"Batch size: {config.BatchSize}");

try
{
    await pipeline.ExecuteAsync();
    log.Info("Pipeline completed successfully");
}
catch (Exception ex)
{
    log.Error("Pipeline failed", ex);
    log.Critical("Pipeline critical failure - data may be inconsistent", ex);
}
